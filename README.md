# ระบบขอเลขหนังสือราชการออนไลน์

ระบบนี้สร้างด้วย Google Apps Script + HTML/CSS/JS  
รองรับหลายผู้ใช้ มีคิวในการขอเลขหนังสืออัตโนมัติ ป้องกันการซ้ำกัน และเคลียร์ผู้ใช้เกิน 5 นาที

## การติดตั้ง

1. สร้าง Google Sheet
   - ชื่อ Sheet:
     - `User` (คอลัมน์: หมายเลขโทรศัพท์ | ชื่อผู้ใช้งาน)
     - `Data` (คอลัมน์: วันที่ | เลขหนังสือ | เรื่อง | ถึงหน่วยงาน | กลุ่มงาน | ผู้บันทึก)
     - `Online` (คอลัมน์: หมายเลขโทรศัพท์ | เวลาเข้าสู่ระบบ | สถานะ)
2. สร้าง Google Apps Script Project
3. วาง `Code.gs` ลงใน Project
4. File > New > HTML File > วาง `index.html`
5. Deploy as Web App
6. เปิด URL เว็บเพื่อใช้งาน
